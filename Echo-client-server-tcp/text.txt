----Server:
    Ο EchoServerTCP ανοίγει ένα ServerSocket στη θύρα 1234 και περιμένει
    μια εισερχόμενη σύνδεση από έναν client.


----Client:
    Ο EchoClientTCP συνδέεται στον server και επικοινωνεί μαζί του.


----ServerProtocol
    Ο ServerProtocol επεξεργάζεται τα αιτήματα του client εφαρμόζοντας διάφορες λειτουργίες, όπως μετατροπές κειμένου, 
    κρυπτογραφήσεις κλπ, και επιστρέφει την κατάλληλη απάντηση.


----ClientProtocol
    Ο ClientProtocol διαχειρίζεται την αλληλεπίδραση με τον χρήστη, 
    λαμβάνοντας επιλογές και μηνύματα προς αποστολή στον server και εμφανίζοντας τις απαντήσεις που επιστρέφει ο server.


ΣΤΟ ΣΥΓΚΕΚΡΙΜΕΝΟ project ΟΙ ΕΠΙΛΟΓΕΣ ΕΙΝΑΙ ΟΙ ΕΞΗΣ:
1. Μετατροπή από κεφαλαία σε πεζά.
2. Μετατροπή από πεζά σε κεφαλαία.
3. Κωδικοποίηση κειμένου σε ASCII με χρήση offset.
4. Αποκωδικοποίηση ASCII σε αρχικό κείμενο με χρήση offset.
5. Αναστροφή μηνύματος (mirrored).
6. Επιστροφή της τρέχουσας ημερομηνίας και ώρας.
7. Καταμέτρηση εμφανίσεων ενός γράμματος σε ένα κείμενο.
